Workshops

Git

Git is one of the most important tool in a programmer's toolbox. Software industry, proprietary or open source, rely heavily on git. In this workshop, students learnt how to use Git for their personal projects. After going through the essential commands, we had a look at how you could put your code in the cloud using GitHub.


Chatbot Architecture

Chatbots are on the rise. Startups are building chatbots, platforms, APIs, tools, analytics. Microsoft, Google and Facebook are introducing tools and frameworks and building smart assistants on top of these frameworks. Multiple blogs, magazines, podcasts report on news in this industry, and chatbot developers gather on meetups and conferences. To keep up with the rising trend of chatbots, we organized a talk on chatbot architecture


React Workshop

We organized a workshop on react which was conducted by Nishant Verma, React is a JavaScript library by Facebook for building user interfaces. It is taking the web by storm. Though web and technologies that run the web are subject to constant change, if someone was to bet on one technology that would still be used in 5 years from now, React will be it.


Applied CS with Android

Google’s Applied CS with Android program was designed for university Computer Science students and enhanced various concepts from their current curricular work. The program revisited concepts from Data Structures and Algorithms, as well as Artificial Intelligence. We used Android as a platform for development because it’s modern, has an easy to use graphical interface.


What the heck is UX ?

We organized a workshop on UI/UX at NIT SURAT which was conducted by Nishant Verma , Musixmatch Design Intern who told our GDG community about UI,Material Design,Design sprints and some secrets of building successful apps.
